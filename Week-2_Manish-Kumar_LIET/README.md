# Week-2_Manish-Kumar_LIET

## EV Sales Forecasting ML Pipeline

### Overview

This project implements an end-to-end Machine Learning Pipeline on Electric Vehicle (EV) sales and pricing data. The objective is to analyze sales trends, build predictive models, and forecast future deliveries.

### Tasks Performed

* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Machine Learning Pipeline using Scikit-learn
* Linear Regression Modeling
* Random Forest Regression
* Hyperparameter Tuning using GridSearchCV
* Time Series Forecasting using ARIMA

### Dataset Features

* Year
* Month
* Region
* Model
* Production Units
* Average Price (USD)
* Battery Capacity (kWh)
* Range (km)
* CO2 Saved (tons)
* Charging Stations

### Target Variable

* Estimated Deliveries

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Statsmodels

### Results

* Linear Regression R² Score: 0.9984
* Random Forest R² Score: 0.99917
* Random Forest outperformed Linear Regression and was selected as the best model.
